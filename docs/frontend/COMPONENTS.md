# Components

## Layout
- **Header** – minimal logo + menu
- **AppLayout** – main wrapper

## Common
- **Greeting** – time-based greeting (without heart)
- **WelcomeTitle** – large italic "Welcome home ❤️"
- **SupportingText** – "A little place that's always here..."
- **DailyEncouragement** – random daily message + divider
- **HeroSection** – full-viewport-height rounded container with overlay
- **FooterComfort** – "Take your time. I'm here. ❤️"

## Mood Feature
- **MoodSelector** – grid layout with mood tiles + "I don't know" tile
- **MoodTile** – square selectable mood card (emoji + title)
- **UnknownMoodTile** – wider variant for "I don't know"
