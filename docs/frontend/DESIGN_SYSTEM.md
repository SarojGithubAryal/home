# Design System

## Colors
- Background: #F7F3EE
- Surface: #FFFFFF
- Primary: #C98983 (dark: #B87570)
- Text: #3A342F, secondary #756E67, muted #A39B92
- Border: #E8DED5
- Success: #9CBF9A, Warning: #D8B36A

## Typography
- Greeting: clamp(2rem, 6vw, 2rem) / 600 / 1.2
- Section title: 1.375rem / 600
- Card title: 1.0625rem / 600
- Body: 0.9375rem / 400 / line-height 1.6
- Small text: 0.75rem

## Spacing
- xs: 8px, sm: 12px, md: 16px, lg: 24px, xl: 32px, 2xl: 48px

## Radius
- sm: 12px, md: 20px, lg: 28px, pill: 50px

## Components
- Card: white bg, padding 24px, radius 20px, soft shadow
- Room card: large radius, scale(0.97) on active
- Mood chip: pill, 44px height, active = primary bg white text
- Primary button: pill, 52px height, full width mobile, primary bg
- Secondary button: minimal text
- Clickable text: primary color, no container
- Bottom nav: fixed, 72px, top border

## Responsive
- Mobile-first, container max-width 600px, padding 20px
- Flexible sizing, clamp() where appropriate
